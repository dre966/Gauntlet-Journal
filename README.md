# Gauntlet Database

A web-based tracking system for **Asphalt Legends Unite**, designed to help players log and analyze their Gauntlet race times.

---

### Project Overview
This project serves as a personal practice ground for sharpening full-stack web development skills. It acts as a digital journal for "Gauntlet Mode," allowing racers to track performance across different tracks to identify where they excel and where they need improvement.

### Tech Stack
* **Frontend:** JavaScript (Dynamic UI population and DOM manipulation)
* **Backend:** PHP (Server-side logic and database communication)
* **Database:** MySQL (Storing user accounts, car selections, and race times)

---

### Page Structure
* **`login.php`** – Secure entry point for users to create accounts or log in.
* **`index.php`** – The main hub where users select or modify their car lineup.
* **`table.php`** – Data visualization page that displays race times in either a structured table or block format.
